Initial Setup

1.) Ensure that Microsoft Visual C++ 2015 Redistributable (x86) is installed. This can be downloaded from Microsoft's website.
2.) Set the PATH variable such that Libaries\opencv\x86\vc14\bin is included. Note that it is recommended that the opencv folder is 
    copied to your machine. A restart is required after this step.

Using the Program

1.) Open the command prompt and navigate to where the executable is located.
2.) Start the executable and, as parameters, give it the path to a folder containing the DICOM data and the output folder, in this order.
    NOTE: the folder containing the DICOM data must ONLY contain the DICOM data. Otherwise, an error will be generated in the program.
3.) Hit Enter to run the program
